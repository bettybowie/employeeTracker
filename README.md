# employeeTracker

## Description

This is a command-line application that uses Node.js, Inquirer, and MySQL, to manage a company's employee database.

## Installation

To install necessary dependencies, run:
   
~~~
npm i
~~~

To start the application on command line, run:

~~~
npm start
~~~
